# Static HTML/CSS/JS Project

## Overview
A simple static web project with HTML, CSS, and JavaScript. Features a Ball class for potential animation/physics demos.

## Project Structure
- `index.html` - Main HTML page
- `main.css` - Stylesheets
- `main.js` - JavaScript with Ball class implementation

## Running the Project
The project uses Python's built-in HTTP server to serve static files on port 5000.

```bash
python3 -m http.server 5000 --bind 0.0.0.0
```
